# Directives

This directory contains **Standard Operating Procedures (SOPs)** written in Markdown.

## Format
- **Goal**: What are we trying to achieve?
- **Inputs**: What data/files are needed?
- **Tools**: Which scripts in `../execution/` should be used?
- **Outputs**: What is the expected result?
- **Edge Cases**: known issues or limitations.

See `../AGENTS.md` for more details on the 3-Layer Architecture.
