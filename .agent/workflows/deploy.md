---
description: Deploy the EcoNexo application
---

1. Lint the codebase to ensure code quality advice.
// turbo
2. Run the lint command
```bash
npm run lint
```

3. Build the application for production.
// turbo
4. Run the build command
```bash
npm run build
```

5. (Optional) Start the production server to verify the build.
6. Run the start command
```bash
npm run start
```
